Day of the Week
---------------

The program returns the day of the week for a given date.

### example ###
month: 1  
day: 1  
year: 1900  
Day of week (Sun = 0, ..., Sat = 6): 1

### example ###
month: 4  
day: 25  
year: 2013  
Day of week (Sun = 0, ..., Sat = 6): 4

### example ###
month: 12  
day: 31  
year: 1999  
Day of week (Sun = 0, ..., Sat = 6): 5
