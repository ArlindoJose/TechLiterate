Powers
------

The user enters a base and the number of powers to calculate. The program then prints out these powers.

### example ###
base: 2  
number of powers: 5  
1  
2  
4  
8  
16  
32  

### example ###
Enter the base: 3  
Enter the number of powers: 10  
3^0 = 1.0  
3^1 = 3.0  
3^2 = 9.0  
3^3 = 27.0  
3^4 = 81.0  
3^5 = 243.0  
3^6 = 729.0  
3^7 = 2187.0  
3^8 = 6561.0  
3^9 = 19683.0  

### example ###
Enter the base: 10
Enter the number of powers: 6
10^0 = 1.0
10^1 = 10.0
10^2 = 100.0
10^3 = 1000.0
10^4 = 10000.0
10^5 = 100000.0
